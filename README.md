## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
Using node v16



## Run project by Docker (for develop)
```bash
# With linux
sh ./init_develop.sh
```

## Run project
### 1.Prepare
```bash
$ cp .env.example .env
```
### 2. Config info in file .env

### 3. Installation
```bash
$ yarn
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# debug mode
$ npm run start:debug

# production mode
$ npm run start:prod
```
